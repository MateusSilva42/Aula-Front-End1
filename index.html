FRONT-END, O QUE É?

O que é desenvolvimento Front-End?

Desenvolvimento front-end é a programação da parte "visual" de uma aplicação, criando a interação entre usuário e sistema. É o que permite que usuários sem conhecimento nenhum de programação possa interagir com sistemas, clicando em botões, preenchendo campos, vendo uma animação bacana de loading enquanto a parte chata acontece por debaixo dos pansos etc.

O que um profissional de front-end faz exatamente?

Um profissional de de front-end é responsável pelo desenvolvimento dessa parte visual, é quem dá vida a páginas estáticas, responsável por deixar tudo no lugar correto, da melhor maneira possível para o usuário,criar sistemas a prova de estupidez que acolhem o usuário da forma mais intuitiva possível.

Principais Ferramentas utilizadas no front end:

Atualmente as principais ferramentas que podem ser usadas por um desenvolvedor front-end incluem, mas não se limitam à:

  • HTML - Linguagem de marcação para estruturar as páginas
  • CSS - Folha de estilo em cascata, usada para dar uma corzinha e estilizar as páginas sem graça em HTML
  • Javascript - Linguagem de programação para tornar uma página estática em uma página funcional de verdade
  • Bootstrap - OU qualquer outra biblioteca que dão uma forcinha ao usar CSS e Javascript
